Boost C++ Library !

Jan 03, 2022

BOOST C++

Easy install in your linux system

Boost is a set of C++ libraries.

The Boost project provides free peer-reviewed portable C++ source libraries.

The project website is www.boost.org, where you can obtain more information and download the current release.

To install it we can use a PPA repository (https://launchpad.net/~mhier/+archive/ubuntu/libboost-latest).

$ sudo add-apt-repository ppa:mhier/libboost-latest
$ sudo apt-get install libboost1.74-dev
